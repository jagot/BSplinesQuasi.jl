    Differentiating functions

    We proceed as indicated in Solving equations:

    julia> function find_second_derivative(B, f::Function)
           S = B'B
           D = Derivative(axes(B,1))
           ∇² = B'D'D*B

           # Project function onto B-spline basis
           cf = B \ f
           # Find derivative
           cg = S \ ∇²*cf

           cf,cg
       end
